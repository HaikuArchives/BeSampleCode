BeOS Sample Code: MIDI Kit

The MIDI Kit allows you to handle MIDI data and communicate with MIDI devices and and other MIDI applications.

For more information, see the corresponding category in the online Developer Library.

Projects

PatchBay A tool that displays a matrix of all the globally-visible MIDI producers and consumers in your system. Use PatchBay to connect and disconnect these MIDI objects.
Transposer A simple MIDI filter that transposes incoming MIDI events.
