BeOS Sample Code: Screen Savers

The BeOS screen saver has a powerful add-on interface, enabling easy development of sophisticated visual displays to occupy the screen while no-one is using it. The examples in this section illustrate the use of various aspects of the screen saver API.

Projects

Ball A simple OpenGL bouncing ball screen saver, without support for hardware acceleration.
Spots The "Spots" saver provided with BeOS.
WhackLoader A more sophisticated saver which maintains persistent configuration state. This saver relies on the Whack example, provided in the game kit sample code area.
