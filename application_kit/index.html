BeOS Sample Code: Application Kit

The Application Kit identifies your application as a distinct entity, and provides a messaging and communication service that lets your application communicate with other applications.

For more information, see the corresponding category in the online Developer Library.

Projects

AppKitModel Shows two different models you can follow when making use of loopers and handlers.
BeBounce How two windows can use messages to communicate with each other.
colorPickerModule Demonstrates a protocol for dragging and dropping colors from one application to another.
Container A simple replicant container.
EZLauncher How to use the application roster to find and launch applications.
Magnify The source for Magnify, one of the BeOS's included applications.
Pulse Another application provided with BeOS, this project shows how to use replicants and the Deskbar.
Scripter Shows how to send scripting messages to an application, window, or view.
ShelfInspector Displays the contents of a shelf -- useful for debugging replicants.
Thesaurus Another example of using scripting messages to communicate between applications.
Watcher Uses the application roster to keep track of currently running applications and mounted volumes.
