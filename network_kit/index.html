BeOS Sample Code: Network Kit

The Network Kit contains C functions that let you identify remote machines that are connected to the network, and communicate with those machines through the TCP and UDP message protocols.

For more information, see the corresponding category in the online Developer Library.

Projects

FtpClient Uses the R4.5 NetAPI to implement an FTP client shared library
Tank Implements a virtual fish tank that spans the network, using the infamous Distributed Fish Protocol.
VoteOften Use the Network Kit to stuff online ballots!
