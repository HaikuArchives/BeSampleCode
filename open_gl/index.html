BeOS Sample Code: OpenGL®

The BeOS uses the industry-standard OpenGL API from within its GUI architecture to implement high-performance 3D graphics.

For more information, see the corresponding category in the online Developer Library.

Projects

3Dlife Implements the game of Life (the computer model, not the board game!) in 3D.
3Dlife-Voodoo Based on the above Life demo, but with support for hardware-accelerated OpenGL on computers with appropriate 3dfx-based hardware.
GLTeapot This implementation of the classic OpenGL "teapot" example also measures graphics performance in frames per second.
GLTeapot-Voodoo Based on the above Teapot demo, but with support for hardware-accelerated OpenGL on computers with appropriate 3dfx-based hardware.
