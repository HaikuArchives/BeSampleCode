BeOS Sample Code: Translation Kit

The Translation Kit provides a general framework for converting data between different formats. It is especially useful for reading and writing images in various formats.

For more information, see the corresponding category in the online Developer Library.

Projects

ppm A Translation Kit add-on that reads and writes PPM image files.
translate Shows how to query the Translator Roster to get information about all of the supported data conversions in your BeOS system.
