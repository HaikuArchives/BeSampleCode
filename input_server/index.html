BeOS Sample Code: Input Server

The input server handles the low-level interface to input devices such as keyboards, mice, writing tablets, or other mechanisms. You can extend the input server to handle your own devices, specify your own input events, and define your own input UI methods.

For more information, see the corresponding category in the online Developer Library (currently under construction).

Projects

Nervous An input server device that generates random mouse move events.
InputRecorder A set of applications for recording and replaying input events such as mouse movements and keypresses.
